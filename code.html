dcf.admin.load_metadata_versions(modeladmin, request, queryset)[source]¶

This admin function (re)loads all of the dcf.models.MetadataModel`s into a :class:`dcf.models.MetadataVersion. For each version in the queryset, it checks all models in the same application. If a model is a subclass of dcf.models.MetadataModel, it adds that model’s name to the list of model_names stored by the dcf.models.MetadataVersion.

Note

During a MetadataVersion’s dcf.models.MetadataVersion.__init__() function, that list of names is used to create a dictionary of the actual model classes themselves.

Warning

This function should never be called directly; it is accessible only via the Django Admin.
