esdoc-questionnaire
================

.. image:: _static/esdoc_logo.png
    :align: right

esdoc-questionnaire is a Django Application that generates webforms based on CIM-compatible metadata models and Controlled Vocabularies.  It can publish the CIM documents created using those forms in an ATOM feed available to external consumers.


esdoc-questionnaire is part of the ES-DOC suite of CIM-related tools.
